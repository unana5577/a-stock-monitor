## 🚀 如何使用

### 1. 启动后端服务

```bash
cd ~/Documents/trae_projects/a-stock-monitor
node server.js
```

服务将运行在：`http://localhost:8787`

### 2. 测试 API 接口

```bash
# 测试板块生命周期接口
curl "http://localhost:8787/api/sector/lifecycle?days=60"

# 测试市场总览接口
curl "http://localhost:8787/api/market/overview"

# 测试涨跌家数接口
curl "http://localhost:8787/api/market/breadth"
```

### 3. 在 VidClaw 中查看

1. 打开 VidClaw：`http://localhost:3333`
2. 点击 **Files**
3. 找到 **a-stock-monitor** 文件夹
4. 即可查看所有文件

### 4. 在 OpenClaw 中访问

```bash
# 在 workspace 中已有符号链接
cd ~/.openclaw/workspace/a-stock-monitor
```
