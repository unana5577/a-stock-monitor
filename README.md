# A 股监控（a-stock-monitor）

用于“日常看盘 + 板块轮动 + 新闻聚合”的轻量站点：部署到服务器后，打开域名即可使用。

## 功能

- **看盘概览**：指数/情绪/量能等快速浏览
- **板块监控**：板块历史、动能/资金、轮动与盘中结构跟踪
- **新闻聚合**：按类型/行业/情绪分类汇总，支持热度统计
- **本地缓存**：数据落盘到服务器本地文件，降低重复抓取与接口延迟

## 技术栈

- Node：提供 API，并静态托管前端页面
- Python：基于 akshare 拉取/计算数据
- 存储：默认仅使用本地文件缓存（无需数据库）

## 快速开始（本地）

```bash
npm install
pip install -U pip
pip install akshare pandas numpy
PORT=8787 node server.js
```

打开：`http://localhost:8787/`
