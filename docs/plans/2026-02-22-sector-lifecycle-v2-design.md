> **版本**: v2.0

## 一、概述

### 1.1 目标

为A股量化软件重新设计板块生命周期判断系统，通过**三层判断逻辑**，帮助用户识别板块状态，并给出3-5天的操作建议。

### 1.2 核心逻辑

```
板块位置（参考信息）+ 动能 + 资金行为 = 操作建议
```

**关键原则**：
- **操作建议**只由「动能+资金行为」决定
- **板块位置**作为参考信息，帮助理解板块整体状态

### 1.3 适用范围

| 场景 | 说明 |
|------|------|
| 板块分析 | 当前需求 |
| 个股分析 | 未来复用，阈值单独调整 |
| 指数分析 | 未来复用，阈值单独调整 |

## 二、指标定义

### 2.1 基础指标

| 后台指标 | 前台显示 | 公式 | 含义 |
|----------|----------|------|------|
| Alpha_5 | 5日超额收益 | 板块5日涨幅 - 基准5日涨幅 | 短期相对强弱 |
| Alpha_20 | 20日超额收益 | 板块20日涨幅 - 基准20日涨幅 | 中期相对强弱 |
| Amount_Share | 当日资金关注度 | 板块成交额 / 全市场成交额 | 资金关注程度 |
| Amount_Share_MA5 | 5日资金热度 | 近5日 Amount_Share 均值 | 资金关注趋势 |
| Amount_Share_Change | 资金热度变化 | Amount_Share / Amount_Share_MA5 - 1 | 相对5日均值的增长 |

### 2.2 价格指标

| 后台指标 | 前台显示 | 公式 | 含义 |
|----------|----------|------|------|
| Close | 收盘价 | - | 当日收盘价 |
| Pct | 今日涨跌 | (今收 - 昨收) / 昨收 × 100% | 当日涨跌幅 |
| MA5 | 5日均线 | 近5日收盘价均值 | 短期趋势线 |
| MA20 | 20日均线 | 近20日收盘价均值 | 中期趋势线 |
| MA60 | 60日均线 | 近60日收盘价均值 | 长期趋势线 |

### 2.3 趋势指标

| 后台指标 | 前台显示 | 公式 | 含义 |
|----------|----------|------|------|
| MA5_Slope | 5日均线斜率 | (MA5今日 - MA5_5日前) / 5 | 短期趋势方向 |
| MA60_Slope | 60日均线斜率 | (MA60今日 - MA60_5日前) / 5 | 长期趋势方向 |
| Bias_20 | 偏离程度 | (收盘价 - MA20) / MA20 × 100% | 偏离20日均线的程度 |
| Bias_20_History_Max | 历史最大偏离 | 历史所有 Bias_20 的最大值 | 历史正乖离极值参考 |
| Bias_20_History_Min | 历史最小偏离 | 历史所有 Bias_20 的最小值 | 历史负乖离极值参考 |

### 2.4 辅助指标

| 后台指标 | 前台显示 | 含义 |
|----------|----------|------|
| RS_Change_20 | 强度变化 | Alpha_20 的变化趋势 |
| Amount_Share_High_20 | 20日资金高点 | 近20日资金关注度最高值 |

## 三、动态基准选择

### 3.1 选择逻辑

| 步骤 | 说明 |
|------|------|
| 1 | 获取板块60日收盘价序列 |
| 2 | 获取4个指数（上证、深证、创业板、科创板）60日收盘价序列 |
| 3 | 计算板块与每个指数的皮尔逊相关系数 |
| 4 | 选择相关系数最高的指数作为基准 |

### 3.2 输出字段

| 字段 | 示例 |
|------|------|
| 基准指数 | 创业板 |
| 相关性 | 0.85 |

### 3.3 后续使用

所有 Alpha 计算都基于动态选择的基准。

## 四、三层判断逻辑

### 4.1 第一层：板块位置（参考信息）

**判断依据**：Alpha_20 + Amount_Share_MA5

**目的**：判断板块的成长性和产业趋势状态

| Alpha_20 | Amount_Share_MA5 | 位置判断 | 含义 |
|----------|------------------|----------|------|
| > +8% | > 40% | 高位区 | 板块过热，产业趋势可能见顶 |
| +3%~+8% | 25%~40% | 中高位区 | 板块偏热，产业趋势仍有机会 |
| -3%~+3% | 15%~25% | 中位区 | 板块正常，产业趋势稳定 |
| < -3% | 10%~15% | 低位区 | 板块低估，产业趋势可能有问题 |
| < -8% | < 10% | 冰点区 | 板块被遗忘，无人关注 |

**阈值说明**：以上阈值为初始值，需通过回测调整。

### 4.2 第二层：动能（趋势方向）

**判断依据**：Alpha_5 + MA5_Slope + Close vs MA5

**目的**：判断短期趋势方向和强度

| Alpha_5 | MA5_Slope | Close vs MA5 | 动能判断 |
|---------|-----------|--------------|----------|
| > +3% | > 0 | Close > MA5 | **强势向上** |
| > +3% | < 0 | Close < MA5 | **强势向下** |
| +1%~+3% | > 0 | Close > MA5 | **偏强向上** |
| +1%~+3% | < 0 | Close < MA5 | **偏强向下** |
| -1%~+1% | 任意 | 任意 | **中性震荡** |
| < -1% | < 0 | Close < MA5 | **弱势向下** |
| < -1% | > 0 | Close > MA5 | **弱势反弹** |

**阈值说明**：以上阈值为初始值，需通过回测调整。

### 4.3 第三层：资金行为（短期资金动向）

**判断依据**：Amount_Share_Change + Bias_20 + Pct

**目的**：判断短期资金的进出行为

| 优先级 | 条件 | 资金行为判断 | 含义 |
|--------|------|--------------|------|
| 1 | Pct < -3% + Amount_Share > 自身历史80分位 | **恐慌出逃** | 资金不计成本离场 |
| 2 | Amount_Share_Change ≥ 50% + Pct > 0 | **放量启动** | 资金进场 |
| 3 | Amount_Share相对20日高点回落 ≥ 20% | **资金撤退** | 资金逐步流出 |
| 4 | Bias_20 > 8% + Pct > 0 | **加速赶顶** | 短期涨幅过大 |
| 5 | Bias_20 < -8% + Pct < 0 | **超跌反弹** | 可能触底 |
| 6 | 以上都不满足 | **横盘整理** | 正常状态 |

**阈值说明**：
- Amount_Share_Change ≥ 50%：资金关注度相对5日均值增长50%
- 以上阈值为初始值，需通过回测调整

## 五、操作建议矩阵

**核心原则**：操作建议只由「动能+资金行为」决定，板块位置作为参考信息。

### 5.1 操作建议表

| 动能 | 资金行为 | 操作建议 | 说明 |
|------|----------|----------|------|
| **强势向上** | 放量启动 | 持股待涨 | 主升阶段，资金确认 |
| 强势向上 | 加速赶顶 | 分批止盈 | 涨速过快，注意风险 |
| 强势向上 | 资金撤退 | 果断离场 | 量价背离，别犹豫 |
| 强势向上 | 恐慌出逃 | 果断离场 | 突发利空，快速离场 |
| 强势向上 | 横盘整理 | 持有观察 | 中继整理，看后续方向 |
| 强势向上 | 超跌反弹 | 低吸机会 | 强势中的回调 |
| **强势向下** | 任意 | 果断止损/观望 | 补跌或杀跌，不参与 |
| **偏强向上** | 放量启动 | 积极建仓 | 突破信号，值得参与 |
| 偏强向上 | 横盘整理 | 持有 | 健康上涨 |
| 偏强向上 | 资金撤退 | 逐步减仓 | 涨势放缓，注意风险 |
| 偏强向上 | 加速赶顶 | 分批止盈 | 短期涨幅过大 |
| **偏强向下** | 任意 | 观望 | 上涨中继的回调，等待企稳 |
| **中性震荡** | 放量启动 | 轻仓试探 | 可能启动，先试探 |
| 中性震荡 | 横盘整理 | 小仓埋伏 | 等待方向选择 |
| 中性震荡 | 资金撤退 | 观望 | 突破失败 |
| 中性震荡 | 超跌反弹 | 关注企稳 | 可能触底 |
| **弱势向下** | 恐慌出逃 | 坚决回避 | 加速下跌 |
| 弱势向下 | 资金撤退 | 空仓观望 | 持续流出 |
| 弱势向下 | 横盘整理 | 观望等待 | 底部磨底 |
| **弱势反弹** | 放量启动 | 关注企稳 | 底部反转信号 |
| 弱势反弹 | 横盘整理 | 等待确认 | 需要更多确认信号 |

### 5.2 板块位置的参考意义

| 板块位置 | 参考意义 |
|----------|----------|
| 高位区 | 心里有数：位置高了，止盈要快，别贪 |
| 中高位区 | 正常偏热，按动能+资金行为操作 |
| 中位区 | 正常状态，按动能+资金行为操作 |
| 低位区 | 可能有机会：反弹空间大，但注意"价值陷阱" |
| 冰点区 | 长期布局机会，短期不一定有机会 |

## 六、输出格式

### 6.1 表格字段

| 字段 | 示例 | 说明 |
|------|------|------|
| 板块名称 | 半导体 | - |
| 基准指数 | 创业板 | 动态选择 |
| 相关性 | 0.85 | 与基准的相关系数 |
| 板块位置 | 高位区 | 第一层判断 |
| **20日超额收益** | +3.8% | 中期相对强弱 |
| **5日超额收益** | +1.0% | 短期相对强弱 |
| 动能 | 强势向上 | 第二层判断 |
| 资金行为 | 放量启动 | 第三层判断 |
| 操作建议 | 持股待涨 | 动能+资金行为决定 |
| 归因说明 | 以创业板为基准... | 详细说明 |
| 乖离对比 | 当前偏离 +8%... | 历史对比 |

### 6.2 归因说明格式

**格式**：
```
以{基准}为基准，{动能归因}，{资金行为归因}。
```

**示例**：
```
以创业板为基准，5日超额收益+5%（短期强势），资金热度相对5日均值增长80%（放量进场）。
```

### 6.3 历史乖离率对比

**计算逻辑**：
- Bias_20_History_Max：历史最大乖离率（正值，如 +12%）
- Bias_20_History_Min：历史最小乖离率（负值，如 -8%）

**输出格式**：

| 场景 | 当前偏离 | 历史极值 | 输出示例 |
|------|----------|----------|----------|
| 正乖离，接近极值 | +11% | +12% | 当前偏离 +11%，历史极值 +12%，接近历史高位，注意回调风险 |
| 正乖离，未达极值 | +8% | +12% | 当前偏离 +8%，历史极值 +12%，未达极值，可能还有上涨空间，但也注意回调风险 |
| 负乖离，接近极值 | -7% | -8% | 当前偏离 -7%，历史极值 -8%，接近历史低位，关注反弹机会 |
| 负乖离，未达极值 | -4% | -8% | 当前偏离 -4%，历史极值 -8%，未达极值，仍有下跌风险 |

**判断逻辑**：
```
if 当前偏离 > 0:
    对比历史最大值（Bias_20_History_Max）
    输出: 上涨空间 + 回调风险提示
elif 当前偏离 < 0:
    对比历史最小值（Bias_20_History_Min）
    输出: 下跌风险 + 反弹机会提示
```

## 七、回测复核功能

### 7.1 阈值回测方案

| 步骤 | 说明 |
|------|------|
| 1 | 遍历历史数据（从2025年5月19日开始） |
| 2 | 对每个时间点判断位置+动能+资金行为 |
| 3 | 记录后续3-5日的涨跌幅 |
| 4 | 计算预测准确率 |
| 5 | 调整阈值，重复步骤1-4 |
| 6 | 选择准确率最高的阈值组合 |

### 7.2 前台复核界面

| 功能 | 说明 |
|------|------|
| 日期切换 | 可以回到历史任意日期，查看当天的判断结果 |
| 对照K线 | 点击日期后，显示该板块的K线图 |
| 标记正确/错误 | 用户可以标记该判断是否正确 |
| 统计准确率 | 自动统计各判断的准确率 |

## 十一、数据规则

### 11.1 数据获取规则

| 场景 | 数据类型 | 操作 |
|------|----------|------|
| 历史数据 | 日线级别 | 直接使用已保存的日线数据 |
| 当日开盘中 | 分时数据 | 实时下载分时数据 |
| 当日收盘后 | 日线数据 | 请求日线数据并保存 |

### 11.2 时间计算规则

| 规则 | 说明 |
|------|------|
| 60日 | 过去60个**交易日**，不是自然日 |
| 20日 | 过去20个**交易日**，不是自然日 |
| 5日 | 过去5个**交易日**，不是自然日 |

**示例**：
- 2026年2月22日往前60个交易日 ≈ 2025年11月中旬（排除周末和节假日）

### 11.3 数据范围

- **历史数据起始**：2025年5月19日
- **回测范围**：2025年5月19日至今
