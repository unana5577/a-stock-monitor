## 信号定义

### 最新信号（2026-02-23 验证）

| 信号 | 含义 | 判断条件 | 准确率 | 状态 |
|------|------|----------|--------|------|
| **long** | 做多 | 板块信号 + 无极端风险 | **68.9%** | ✅ 可用 |
| **false_kill** | 错杀机会 | 上升期 + 大盘恐慌 | **57.7%** | ✅ 可用 |
| ~~avoid~~ | ~~回避~~ | ~~已废弃~~ | 41.8% | ❌ 已废弃 |
| **neutral** | 观望 | 主线不明/轮动/盘整 | - | - |

### 信号定义说明

**long**：
- 板块处于上升期或启动期
- 没有极端市场风险（如大盘恐慌、政策利空）
- 适合建仓或持有

**false_kill**：
- 板块处于上升期
- 大盘出现恐慌性下跌
- 板块被错杀，反而是买点

**neutral**：
- 主线不明确
- 板块快速轮动
- 大盘盘整期

## 回测脚本配置

### 核心文件
```
backend/
├── backtest_thresholds.py      # 回测脚本
├── sector_lifecycle.py         # 核心判断逻辑
├── sector_lifecycle_config.py   # 阈值配置
└── fetch_sector_data.py        # 数据获取
```

### 回测脚本功能
```python
# backtest_thresholds.py

主要功能：
1. 加载历史数据（2025-05-19起）
2. 应用板块生命周期判断逻辑
3. 输出操作建议（long/neutral/false_kill）
4. 验证未来5日涨跌
5. 计算准确率、胜率、夏普比率
6. 输出统计报告
```

### 回测输出
```
- 总体准确率
- 各信号准确率
- 各板块准确率
- 胜率
- 平均收益
- 夏普比率
- 最大回撤
```
