## 归类规则

### 国家优先级
**美国 > 日本 > 中国**

### 类型分类

| 类型 | 包含内容 |
|------|----------|
| **宏观** | GDP、CPI/CCI、利率、央行政策、财政政策、汇率、十五五规划重点项目、国家重大会议（两会、中央经济工作会议等） |
| **地缘** | 战争、制裁、贸易摩擦、外交冲突 |
| **行业** | 具体板块/公司的新闻 |

### 板块标签（sector）
- 半导体
- 云计算
- 新能源
- 商业航天
- 创新药
- 有色金属
- 通讯设备

### 情绪（sentiment）
- 1：利好
- 0：中性
- -1：利空

### 优先级（level）
- P0：最高（美日同时战争、重大黑天鹅）
- P1：重点提示（美国CPI/CCI、中国重大会议、十五五规划重大政策）
- P2：一般重要
- P3：普通新闻

### 重点提示条件
1. 美日同时有战争 → P0
2. 美国 CCI/CPI 数据 → P1
3. 美国重要政策/声明 → P1
4. 中国重大会议（两会、中央经济工作会议）→ P1
5. 十五五规划重大政策 → P1

---

## 筛选规则

### 保留
- 与 A 股直接相关的新闻
- 标题/内容包含：股票、板块、指数、政策、公司名

### 丢弃
- 纯广告、软文
- 与市场无关的社会新闻
- 营销号内容

---

## 去重规则

1. **URL 去重**：同一链接直接去重
2. **标题相似度**：相似度 > 80% 视为重复，保留最早的
3. **时间窗口**：同一事件 24 小时内只保留第一条

## 数据格式规范（Trae 要求）

### 存储格式
```
data/news/{YYYY-MM-DD}.json → 纯数组
```

### 每条新闻字段
```json
{
  "news_id": "唯一标识",
  "title": "标题",
  "content": "内容（优先）或 summary",
  "source": "来源",
  "url": "原文链接",
  "classify": {
    "type": "宏观/地缘/行业",
    "sector": "板块",
    "sentiment": 1/0/-1,
    "level": "P0/P1/P2/P3"
  }
}
```

### API 响应
```
GET /api/news?date=2026-02-24&sector=半导体
GET /api/news/heat
```
