**优点**:
- ✅ 实时同步（Trae 更新，股票项目自动看到）
- ✅ 不占用额外空间
- ✅ 两个项目共享数据

**操作**:
```bash

# 交易时间每 5 分钟更新分时数据
*/5 9-15 * * 1-5 cd ~/Documents/trae_projects/a-stock-monitor && python3 fetch_minute_data.py
```

**2. 添加错误重试**
```python
import time

def fetch_with_retry(fetch_func, max_retries=3):
    for i in range(max_retries):
        try:
            return fetch_func()
        except Exception as e:
            if i == max_retries - 1:
                raise
            time.sleep(5)
```
