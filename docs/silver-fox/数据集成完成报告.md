## 🚀 如何使用

### 在 Python 中使用

```python
from data_reader import read_minute_data, read_daily_data, read_market_breadth

# 1. 读取今天的分时数据（上证指数）
minute_data = read_minute_data("2026-02-24", "sse")
print(f"获取到 {len(minute_data)} 条分时数据")

# 2. 读取日线数据
daily_data = read_daily_data("2026-02-24")
print(f"上证指数历史数据: {len(daily_data['series']['sse'])} 条")

# 3. 读取涨跌家数
breadth = read_market_breadth("2026-02-24")
print(f"上涨: {breadth['up']}, 下跌: {breadth['down']}")
```

### 在 VidClaw 中查看

1. 打开 VidClaw：http://localhost:3333
2. 进入 **Files** → **股票项目** → **data** → **trae-data**
3. 即可看到所有数据文件
