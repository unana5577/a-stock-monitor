3. 修改文件名生成逻辑：
   ```python
   if args.mode == 'realtime':
       now = datetime.now()
       if now.hour < 9:
           filename = 'pre_market.json'
       else:
           time_slot = f"{now.hour:02d}:{(now.minute // 10) * 10:02d}"
           filename = f"{time_slot}.json"
   else:
       filename = f"{date_str}.json"
   ```
