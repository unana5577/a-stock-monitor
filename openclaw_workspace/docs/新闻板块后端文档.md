## 🔧 脚本说明

### 1. fetch_news.py - 新闻抓取

**功能**: 使用 akshare 获取 A股新闻数据

**执行方式**:
```bash
cd /Users/una5577/Documents/trae_projects/a-stock-monitor/openclaw_workspace/backend/
python3 fetch_news.py
```

**数据保存路径**: 
```
/Users/una5577/Documents/trae_projects/a-stock-monitor/data/news/
```

**输出格式**:
```json
{
  "date": "2026-02-25",
  "fetch_time": "2026-02-25T13:00:16",
  "source": "akshare/stock_news_main_cx",
  "total": 100,
  "news": [
    {
      "news_id": "abc123",
      "title": "瑞银发布锂市场将出现实质性短缺",
      "content": "近日瑞银报告发表观点，称2026 年全球锂市场将出现实质性短缺...",
      "source": "东方财富",
      "url": "https://...",
      "tag": "产业经济"
    }
  ]
}
```

**标题提取规则**:
- 最多20字
- 保留重要机构（瑞银、高盛等）
- 提取核心主体（锂市场、韩国股市等）
- 保留冲击力词（大关、里程碑等）
- 例：`"瑞银发布锂市场将出现实质性短缺"`、`"韩国股市突破5,000点大关"`

## 🌐 API 接口

### GET /api/news

**参数**:
- `date`: 日期 (YYYY-MM-DD) - 必填
- `sector`: 板块筛选 - 可选
- `level`: 等级筛选 (P0/P1/P2/P3) - 可选
- `limit`: 返回数量 (默认50，最大500) - 可选

**示例**:
```bash
# 获取今日新闻
curl "http://localhost:8787/api/news?date=2026-02-25"

# 筛选半导体板块
curl "http://localhost:8787/api/news?date=2026-02-25&sector=半导体"

# 筛选 P0 级新闻
curl "http://localhost:8787/api/news?date=2026-02-25&level=P0&limit=10"
```

**响应**:
```json
{
  "date": "2026-02-25",
  "total": 55,
  "filtered": 55,
  "news": [...]
}
```
