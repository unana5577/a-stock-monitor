## 🆕 新增需求（2026-02-24 22:38）

### 1. 新闻实时更新机制

#### 时间划分
- **开盘前数据**：9:00 之前获取的新闻
- **开盘后数据**：9:00-15:00 获取的新闻

#### 更新频率
- **交易时间**（9:00-15:00）：每 10 分钟更新一次
- **非交易时间**：不更新

#### 数据存储
```
data/news/
├── 2026-02-24/
│   ├── pre_market.json      # 开盘前数据（9:00前）
│   ├── 09:00.json           # 9:00-9:10
│   ├── 09:10.json           # 9:10-9:20
│   ├── 09:20.json           # 9:20-9:30
│   ├── ...
│   ├── 14:50.json           # 14:50-15:00
│   └── daily_summary.json   # 当日汇总
```

#### 定时任务设置
```bash
# 编辑 crontab
crontab -e

# 添加以下行（交易时间每10分钟更新）
*/10 9-15 * * 1-5 cd ~/Documents/trae_projects/a-stock-monitor && python3 fetch_news.py --mode=realtime

# 每日收盘后汇总
0 16 * * 1-5 cd ~/Documents/trae_projects/a-stock-monitor && python3 classify_news.py --mode=summary
```

### 3. 新闻数据记录（方便回测）

#### 数据库设计
```sql
-- 新闻主表
CREATE TABLE news (
    news_id VARCHAR(32) PRIMARY KEY,
    title TEXT NOT NULL,
    content TEXT,
    source VARCHAR(50),
    url TEXT UNIQUE,
    publish_time DATETIME,
    crawl_time DATETIME,
    
    -- 分类字段
    news_type VARCHAR(20),      -- macro/geopolitics/sector
    region VARCHAR(10),         -- CN/US/JP
    sector VARCHAR(50),         -- 板块名称
    sentiment INT,              -- 1/0/-1
    impact_level VARCHAR(5),    -- P0/P1/P2/P3
    
    -- 影响分数
    impact_score FLOAT,         -- [-1, 1]
    
    -- 时间标记
    time_slot VARCHAR(20),      -- pre_market/09:00/09:10/...
    trade_date DATE,            -- 交易日期
    
    -- 元数据
    created_at DATETIME,
    updated_at DATETIME
);

-- 新闻-板块关联表
CREATE TABLE news_sector_impact (
    id INT PRIMARY KEY AUTO_INCREMENT,
    news_id VARCHAR(32),
    sector_code VARCHAR(20),
    impact_direction INT,       -- 1利好/-1利空/0中性
    FOREIGN KEY (news_id) REFERENCES news(news_id)
);

-- 新闻-股票关联表
CREATE TABLE news_stock_impact (
    id INT PRIMARY KEY AUTO_INCREMENT,
    news_id VARCHAR(32),
    stock_code VARCHAR(10),
    is_watched BOOLEAN,         -- 是否关注股票
    FOREIGN KEY (news_id) REFERENCES news(news_id)
);
```

### 4. 类型分类详细说明

#### 宏观新闻（Macro）

**定义**：影响整体市场或多个行业的重大事件

**关键词**：
```python
MACRO_KEYWORDS = [
    # 货币政策
    "央行", "美联储", "利率", "准备金", "降息", "加息",
    "货币政策", "流动性", "MLF", "逆回购",
    
    # 财政政策
    "财政政策", "税收", "财政赤字", "国债",
    
    # 经济数据
    "GDP", "CPI", "PPI", "PMI", "社融", "M2",
    
    # 监管政策
    "证监会", "银保监会", "发改委", "国务院",
    "监管", "政策", "新规", "改革"
]
```

**影响等级**：
| 等级 | 事件类型 | 示例 |
|------|---------|------|
| **P0** | 重大政策转向 | 地产救市、金融改革、系统性风险 |
| **P1** | 重要政策/数据 | 降准/降息、GDP/CPI、监管新规 |
| **P2** | 常规政策调整 | 行业监管、地方政策 |
| **P3** | 常规数据发布 | 专家观点、常规数据 |

#### 地缘政治（Geopolitics）

**定义**：国际关系、战争冲突、贸易摩擦等

**关键词**：
```python
GEOPOLITICS_KEYWORDS = [
    # 战争冲突
    "战争", "冲突", "军事", "制裁", "封锁",
    
    # 贸易摩擦
    "贸易战", "关税", "制裁", "反制",
    
    # 国际关系
    "中美", "美国", "日本", "台海", "南海",
    "朝鲜", "俄罗斯", "欧盟",
    
    # 外交事件
    "访华", "会谈", "外交", "协议"
]
```

**影响等级**：
| 等级 | 事件类型 | 资金情绪影响 | 操作建议 |
|------|---------|-------------|---------|
| **G0** | 大规模战争/中美冲突 | 极端避险，资金外流 | 空仓/降仓至20% |
| **G1** | 局部冲突/重大制裁 | 强避险，板块分化 | 回避敏感板块 |
| **G2** | 贸易摩擦/外交争端 | 短期波动，情绪扰动 | 降低仓位 |
| **G3** | 常规外交事件 | 影响有限 | 正常操作 |

#### 行业新闻（Sector）

**定义**：特定行业的政策、公司、技术等事件

**7个核心板块关键词**：

| 板块 | 关键词 |
|------|--------|
| **半导体** | 芯片、半导体、晶圆、光刻、国产替代、华为、集成电路、GPU、CPU |
| **云计算** | 云计算、数据中心、服务器、AI、大模型、算力、云服务 |
| **新能源** | 电动车、电池、光伏、风电、储能、碳中和、锂电、充电桩 |
| **商业航天** | 卫星、火箭、航天、低空、飞行汽车、卫星互联网 |
| **创新药** | 医药、药品、生物、医保、集采、创新药、疫苗、临床试验 |
| **有色金属** | 铜、铝、锂、稀土、黄金、大宗商品、钴、镍 |
| **通讯设备** | 5G、通信、基站、光通信、光纤、网络设备 |

**影响等级**：
| 等级 | 利空类型 | 利好类型 | 时效性 |
|------|---------|---------|--------|
| **S0** | 行业取缔/龙头暴雷 | 重大政策扶持/技术突破 | 长期（>6个月） |
| **S1** | 监管加强/业绩下滑 | 政策补贴/订单增长 | 中期（1-6个月） |
| **S2** | 价格下跌/竞争加剧 | 产品涨价/份额提升 | 短期（1-4周） |
| **S3** | 个股负面/分析师下调 | 个股利好/分析师上调 | 超短期（<1周） |

### 5. 关注板块和关注股票记录

```python
# 配置文件：config/watched_sectors.json
{
```

### 3. API 接口开发

#### GET /api/news
**参数**：
- `date`: 日期（可选，默认最新）
- `time_slot`: 时间分片（可选：pre_market/09:00/09:10/...）
- `sector`: 板块名称（可选）
- `is_watched`: 是否关注板块（可选：true/false）
- `limit`: 数量限制（默认100）

**返回格式**：
```json
{
    "date": "2026-02-24",
    "time_slot": "pre_market",
    "total": 45,
    "news": [...]
}
```

#### GET /api/news/summary
**功能**：返回当日新闻汇总

**返回格式**：
```json
{
    "date": "2026-02-24",
    "total": 120,
    "by_time_slot": {
        "pre_market": 30,
        "09:00": 10,
        "09:10": 15,
        ...
    },
    "by_sector": {
        "半导体": 20,
        "云计算": 15,
        ...
    },
    "by_sentiment": {
        "利好": 50,
```
